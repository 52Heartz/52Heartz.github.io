# SQL 关键字默认大写

需求：格式化 SQL 的时候自动把数据库关键字转换为大写。

解决方案：

`ctrl` + `alt` + `S` 打开系统设置，进入 `Editor`-`Code Style`-`SQL`-`General`，修改 `Word Case`-`Keywords` 为 `To Upper`。
