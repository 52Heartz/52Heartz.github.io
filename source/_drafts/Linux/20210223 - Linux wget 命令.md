# 安装

## RPM 安装

如果没有 wget，可以使用 curl 下载 wget 的 rpm 包，可以使用网易的镜像：

```
curl -O http://mirrors.163.com/centos/6.2/os/x86_64/Packages/wget-1.12-1.4.el6.x86_64.rpm
```



安装：

```
rpm -ivh wget-1.12-1.4.el6.x86_64.rpm
```
