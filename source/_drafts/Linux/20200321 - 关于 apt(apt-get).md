全名 `Advanced Package Tool`。是由 `The Debian Project` 开发的。

> Debian 使用一套名为 [Advanced Packaging Tool](https://wiki.debian.org/Apt)（APT）的工具来管理这种包系统，不过请不要把它与 apt 命令混淆，它们之间是其实不是同一个东西。



# 命令

## 安装

```bash
sudo apt install [package-name]
```
