写法1：

```sql
SELECT *
FROM table_1
LIMIT 20, 5800;
```



改善后的写法：

```sql
SELECT *
FROM table_1
WHERE id NOT IN (SELECT id
                 FROM table_1
                 ORDER BY id
                 LIMIT 0)
ORDER BY id
LIMIT 20;
```
