LeetCode 154. Find Minimum in Rotated Sorted Array II。

<!-- more -->

本题和第 153 题可以使用同样的方法解决。

# 解答

```java
class Solution {
  public int findMin(int[] nums) {
        if (nums.length == 0) {
            return 0;
        }

        int min = nums[0];

        for (int i = 1; i < nums.length; ++i) {
            if (nums[i] < min) {
                return nums[i];
            }
        }

        return min;
  }
}
```
