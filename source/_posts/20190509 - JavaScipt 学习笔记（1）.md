# 编写和运行

直接把 JavaScipt 代码编写在一个 HTML 文件中，然后直接使用浏览器打开这个 HTML 文件即可运行相应的 JavaScript 代码。

把 JavaScript 代码放在 HTML 文件中的 `<script></script>` 标签中。



## 小测试

```html
<!doctype html>
<html>
<head>
    <title>JS-1</title>
    <script>
        setTimeout(wakeUpUser, 1000);
        function wakeUpUser() {
            alert("Hello!")
        }
    </script>
</head>
<body>
    <h1>JS-1 Test</h1>
</body>
</html>
```
