Java SE 8 流库。

<!-- more -->

Java SE 8 中的流库和 Lambda 表达式是同时在 Java SE 8 中引入的内容。两者常常需要配合使用。





# List 转换为数组

```java
List<Integer> list = new ArrayList<>();
list.add(0);
list.add(1);
list.stream().parallel().mapToInt(Integer::intValue).toArray();
```
