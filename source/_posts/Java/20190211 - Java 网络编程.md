# TCP端口

TCP协议规定，使用 `16bit` 的大小存储端口号，因此，端口号的范围是0~65535。

0~1023已经保留给已知的特定服务。1024~65535可以由程序员挑选使用。

## 常见服务使用的端口号

FTP：20

Telnet：23

SMTP：25

Time：37

HTTP：80

POP3：110

HTTPS：443
